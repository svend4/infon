# Network Package

P2P networking via Yggdrasil mesh network.

## Responsibilities

- Embedded Yggdrasil node
- UDP media transport
- Congestion control
- NAT traversal
- Connection management

## Files

- `yggdrasil.go` — Yggdrasil integration
- `transport.go` — UDP media transport
- `congestion.go` — Adaptive bitrate control
- `peer.go` — Peer connection management
